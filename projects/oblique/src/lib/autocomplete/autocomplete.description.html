The ObAutocompleteInput feature offers an input text field with a filterable autocomplete and highlighting of the matched text in options.

Highlight text by regex

For more specific criteria for highlighting use filterRegexPattern and filterRegexFlag.

filterRegexPattern should include the string 'textToFind' that marks the location where the entered input text will be.

Examples:

filterRegexPattern is (textToFind) it highlights everything enclosed in the input text.

filterRegexPattern is [textToFind]{2} highlights everything that has 2 of the characters in minimum of the input text.
