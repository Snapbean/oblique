Material Design proposes a lot of button variants, but for consistency reasons, Oblique limits this selection to just three alternatives. For that, Oblique proposes a directive, that, when used in conjunction with the matButton directive, applies the correct styles to the underlying button. The three variants are intended to reflect the importance of the button and are named primary, secondary and tertiary. Oblique only styles buttons that are marked with the obButton directive.

Note that the obButton is meant to be used with the mat-button directive only. While material's other button variants are still possible, it is not advised to use them as this could lead to undesirable effects. Furthermore, the ObButton automatically sets the color input to primary, meaning that it should not be manually set.
