The angular material Datepicker has been slightly restyled to match Oblique's look & feel.

IE11 & Edge

For the datepicker to be correctly displayed on both IE11 and Edge 44 (the one without Chromium), the ms-fix class has to be added to the mat-form-field element. Without this class, the datepicker's height will be too large.

Wrapper for the ng-bootstrap Datepicker component with some controls (clear, open & close). The validation is left to the containing form.

Do NOT apply form-control class on or-date-picker as it will break the layout.

Deprecated with Material Design

This component is only meaningful with Bootstrap theme. As a partially deprecated component, it won't be prioritized for new features or improvements but won't be removed as long as Bootstrap is supported.
