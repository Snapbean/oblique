A transparent directive for external links that aims at improving both the security and the accessibility of such links. The default values can be configured through the EXTERNAL_LINK injection token. This directive is added automatically to every a tag and enhances those with an href value that does not match the current url.

Accessibility

Accessibility is enforced by opening the link in a new tab while providing a visual hint in the form of an icon, and a text for screen readers. The position of the icon is configurable.

Security

Security is enforced by adding the rel attribute with the value noopener noreferrer to hide the source page from the destination.
