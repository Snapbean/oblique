Adds control-mandatory class to highlight required elements. Also controls the has-error class to style the component in case of errors.

Required elements will be discovered by either required attribute on the related form control, mandatoryinput on the directive or ngControl.errors.required error on the related form control.

Works well with the or-error-messages component which displays the errors.

Deprecated

This component is meant to be used with Bootstrap only. Material theme should instead use the Angular implementation.

Deprecated with Material Design

This component is only meaningful with Bootstrap theme. As a partially deprecated component, it won't be prioritized for new features or improvements but won't be removed as long as Bootstrap is supported.
