The Multiselect component allows you to select multiple elements, using checkboxes in a dropdown menu.

In case the model is an object, the available options (i.e. content of dropdown) will be defined by:

  1. labelFormatter
  2. labelProperty
  3. model.toString()

And the selected options by:

  1. titleFormatter
  2. titleProperty
  3. labelFormatter
  4. labelProperty
  5. model.toString()

Deprecated with Material Design

This component is only meaningful with Bootstrap theme. As a partially deprecated component, it won't be prioritized for new features or improvements but won't be removed as long as Bootstrap is supported.

While an ObMultiselect component do exists, it is only meant for Bootstrap theme. Projects that do use Material Design should instead use the mat-select component that is documented below.
