A toggleable absolute position container to display supplementary info like help. It consists of a sidebar with a dark background that slides in from the right and takes up 40% of the available space.

The content of the sidebar has to be placed into a container with ob-off-canvas-sidebar class that is a sibling to another container with the ob-off-canvas-main. The parent of both these container needs the ob-off-canvas class. Then, the obOffCanvasToggle directive, that can be applied anywhere, will control the state of the sidebar.
