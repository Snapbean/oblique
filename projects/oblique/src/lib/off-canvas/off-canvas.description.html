A toggleable absolute position container to display supplementary info like help. It consists of a sidebar with a dark background that slides in from the right and takes up 40% of the available space.

The content of the sidebar have to be placed into a container with offcanvas-sidebar class that is siblings to another container with the offcanvas-main. The parent of both these container needs the offcanvas class. Then, the orOffCanvasToggle directive, that can be applied anywhere, will control the state of the sidebar.
