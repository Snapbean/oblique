ObPaginatorService is a customized wrapper from MatPaginatorIntl.

It translates mat-paginator labels.

It provides translation for:

  1. The label for the length of each page
  2. The range text displayed to the user
  3. The tooltip messages on the navigation buttons

The used keys are:

  • i18n.pagination.items-per-page
  • i18n.pagination.next-page
  • i18n.pagination.previous-page
  • i18n.pagination.of-label
  • i18n.pagination.first-page
  • i18n.pagination.last-page

This keys are included in Oblique's translation files and could be overwritten
