A popover provides helpful contextual information in a dedicated container. It is meant for longer descriptions, help or additional actions that are not critical to the application.

The template is rendered at the end of the body to avoid z-index problems and is even removed from the DOM when not active. Its accessibility is ensured through the aria-describedby attribute and the tooltip role. The positioning is achieved with the popper library, for more information see popper.js.org.
