Use a custom JSON schema in order to perform automatic form control validation.

The service support JSON schema draft07 out of the box. Older drafts need some configuration of the underlying plugin, please see the relevant documentation. Note that Draft03 is only partially supported.

Content-Security-Policy

If your project uses a CSP Policy, please note that the Schema Validation needs the unsafe-eval directive on script-src to be activated. This is a security downgrade that cannot be avoided. We are searching for a solution that doesn't require the unsafe-eval directive.
