The sticky component is not to be confused with the position: sticky CSS rule. It simply provides a scrolling container with optional fixed header and footer. This means the scrollbar will be applied to the <ng-template #obStickyMain> part while both <ng-template #obStickyHeader> and <ng-template #obStickyFooter> remain fixed.

For the scrollbar to appear, the ObStickyComponent needs to be higher than its parent, which implies this component must be used within a fixed height context. For example, it can be used to add a fixed header and footer in a ObColumnLayoutComponent with the ob-full-height class. Or within the ObMasterLayoutComponent with sticky header and footer.
