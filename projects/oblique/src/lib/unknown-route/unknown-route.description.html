A simple unknown route fallback component.

Import UnknownRouteModule in the application's root routing module, i.e. either AppRoutingModule or AppModule in a regular CLI project. Then add the following route as the last one {path: '**', redirectTo: 'unknown-route'}.

Per default, the UnknownRouteComponent redirects to /home. This can be tweaked in the MasterLayoutConfigService: 

export class AppModule {
	constructor(config: ObMasterLayoutConfig) {
		config.homePageRoute = '/oblique';
	}
}
