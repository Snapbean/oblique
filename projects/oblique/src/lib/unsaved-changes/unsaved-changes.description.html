Tracks and notifies about unsaved form changes and ask for confirmation in case of:

  • navigating to another route
  • reloading the current browser tab
  • closing the current browser tab
  • closing the browser itself

The custom message shown in the popup asking for confirmation can be changed by overriding i18n.validation.unsavedChanges. It is only shown when navigating to another route but not when leaving or reloading the page. Then the default message of the browser is shown.
